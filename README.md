# binUi

根据个人对html及css开发习惯，开发的一套自己的移动端基础ui组件

# 开发

```bash
npm install
```

```bash
gulp watch
```

# 使用

```bash
<link rel="stylesheet" href="../bulid/css/binui.min.css"> 
```
